About

We are try to build a free online guide book for climbers.

This site is inspirited by Wikipedia, all the contents are contributed by users, anyone can read, edit routes information on the fly. We encourage climbers to share the latest route informations(topos, location, environment etc) on the site after visiting them, that would be very helpful for other climbers.

The source code of this site is accessable on GitHub, welcome to contribute!
